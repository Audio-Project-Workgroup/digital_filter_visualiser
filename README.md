## Download
- run `git clone --recurse-submodules https://github.com/Audio-Project-Workgroup/digital_filter_visualiser.git && cd digital_filter_visualizer`

## Building
Requirements:
- Cmake

### Windows
- run `build`

### Mac & Linux
- run `./build.sh`
